### 插入排序原理描述

* 利用插入法对无序数组排序时，将数组arr划分为两个子区间arr[0..i-1]--已经排好序的有序区和arr[i..n]--当前未排序部分，称为无序区；
* 插入排序的基本操作时将当前无序区的第1个元素插入到有序区中适当的位置上，使arr[0..i]变为新的有序区，这种方法每次使有序区增加1个记录，通常称增量法；
* **无序区元素插入有序区时，比较从有序区由后向前依次比较**；
